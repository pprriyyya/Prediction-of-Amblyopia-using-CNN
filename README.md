👁️ Predicting Amblyopia in Early Stage Using CNN

Harnessing Deep Learning and Brain Connectivity Analysis for Early Detection of Amblyopia

🧠 About the Project

Amblyopia, commonly known as Lazy Eye, is one of the leading causes of vision impairment in children. Early diagnosis is critical because delayed treatment can lead to permanent vision loss.

This project combines Functional Magnetic Resonance Imaging (fMRI) and Convolutional Neural Networks (CNNs) to detect neural abnormalities associated with amblyopia before significant visual symptoms appear.

Instead of relying solely on traditional eye examinations, our approach analyzes brain functional connectivity patterns, enabling a more objective and data-driven diagnosis.

🚀 Key Highlights

✨ Early-stage amblyopia prediction using brain activity data

🧠 Functional Connectivity (FC) Matrix generation from fMRI scans

🤖 CNN-based deep learning architecture for automated classification

📊 Comprehensive performance evaluation using standard metrics

🔬 AI-powered healthcare solution for future diagnostic systems
